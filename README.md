# Customer Churn Analysis Dashboard (Power BI)

##  Overview

This project analyzes customer churn behavior using Power BI to identify key factors influencing customer retention and business performance.

---

## Objectives

* Identify key drivers of customer churn
* Analyze churn across contract types, tenure, and payment methods
* Provide actionable insights to improve customer retention

---

## Tools & Technologies

* Power BI
* Power Query
* DAX
* Excel

---

## Key Insights

* 43% churn observed in month-to-month contracts
* 53% of churn occurs within the first 6 months
* Customers using electronic check show highest churn (~45%)
* Customers without tech support have higher churn rates
